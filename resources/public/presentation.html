Введение в Оценку Ядерной Плотности
(Kernel Density Estimation, KDE)

Основные Идеи:

Оценка ядерной плотности (KDE) представляет собой метод в статистике, который используется для аппроксимации функции плотности вероятности случайной величины на основе наблюдаемых данных. Этот метод часто применяется в визуализации данных и анализе распределений.

  • Ядро (Kernel): В основе KDE лежит понятие “ядра”. Ядро представляет собой функцию, которая помещается вокруг каждой точки данных и служит для сглаживания распределения.

  • Сглаживание: KDE позволяет получить более гладкое представление распределения данных, избегая строгих предположений о его форме. Это особенно полезно, когда форма распределения неизвестна.

Как это работает:

  • Генерация Ядер: Для каждой точки данных создается копия ядра. Чем ближе точки, тем больше ядра влияют на конечную оценку.

  • Суммирование Ядер: Ядра суммируются для всех точек данных, создавая общую оценку плотности вероятности.

  • Настройка Параметров: Один из важных параметров - ширина ядра или “ширина полосы” (bandwidth). Этот параметр регулирует, насколько далеко влияют ядра от каждой точки данных.

Если кратко:

Оценка ядерной плотности строит вокруг каждой точки данных ядро, которое может представлять собой, например, нормальное распределение (гауссовское ядро). Ядро служит весовой функцией, определяющей, насколько каждая точка данных влияет на окружающие ее области при построении оценки плотности.

Применение на практике:

KDE может использоваться для:

  • Визуализации формы распределения данных.

  • Поиска моды (пиков) распределения.

  • Аппроксимации функции плотности вероятности для статистического анализа.

Использование гауссовского ядра (для нашего случая):

\[ K(x, x_i, h) = \frac{1}{\sqrt{2\pi}} \exp\left(-\frac{1}{2}\left(\frac{x - x_i}{h}\right)^2\right) \]

\(K(x,x_i,h) \text{ - гаусовское ядро}\)

\(x \text{ - переменная, для которой рассчитывается плотность}\)

\(x_i \text{ - точка данных}\)

\(h \text{ - ширина полосы}\)

\(\frac{1}{\sqrt{2\pi}} \text{ - нормализующий коэффициент, который гарантирует, что площадь под функцией} \\ \text{гауссовского ядра равна 1 при интегрировании по всем возможным значениям } x. \\ \text{Этот коэффициент обеспечивает корректность интерпретации плотности вероятности.}\)

Использование гауссовского ядра (для нашего случая):

\(\textbf{Нормализующий Коэффициент:}\) \(\frac{1}{\sqrt{2\pi}}: \text{Это нормализующий коэффициент, гарантирующий, что общая площадь под гауссовским ядром равна 1.}\\ \text{Он стандартизирует ядро, чтобы получить правильную функцию плотности вероятности (PDF).}\)

Использование гауссовского ядра (для нашего случая):

\(\textbf{Экспоненциальная ф-ия:}\) \(\exp\left(-\frac{1}{2}\left(\frac{x - x_i}{h}\right)^2\right): \text{Этот термин представляет собой экспоненциальную функцию,}\\ \text{примененную к аргументу. Он моделирует убывание влияния точек данных по мере удаления от точки } x.\\ \frac{x - x_i}{h}: \text{Эта часть определяет "расстояние" между переменной } x \text{ и точкой данных } x_i,\\ \text{нормализованное шириной окна } h.\\ \text{Возведение в квадрат и отрицание: Возведение в квадрат гарантирует, что расстояния будут положительными,}\\ \text{что расстояния будут положительными, и отрицание переворачивает}\\ \text{кривую для моделирования убывания по мере увеличения расстояния.}\)

Использование гауссовского ядра (для нашего случая):

\(\textbf{Объединение компонентов ф-ии плотности:}\) \(\frac{1}{\sqrt{2\pi}} \exp\left(-\frac{1}{2}\left(\frac{x - x_i}{h}\right)^2\right): \text{Вся формула объединяет нормализацию и влияние каждой точки данных}\\ \text{для вклада в оценку плотности вероятности в точке } x.\)

Находим среднюю ставку

Для того, чтобы найти среднюю ставку, предпримем следующее:

  1. Найдем значения ставок в пиках распределения (моды)

  2. Присвоим модам вес, чтобы убрать резкое правое смещение

  3. Посчитаем среднее для логнормального распределения вокруг каждой моды

  4. Округлим до натурального числа в большую сторону значения в массиве - ceiling

  5. Присвоим веса значениям в массиве, совпадающим с полученными средними

  6. Найдем общее средневзвешенное
